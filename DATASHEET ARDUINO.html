Visión general

Arduino Uno es un tablero para microcontroladores basado en un microcontrolador ATmega328P de 8 bits. Junto con ATmega328P, consta de otros componentes como el oscilador de cristal, la comunicación en serie, el regulador de voltaje, etc., para soportar el microcontrolador. Arduino Uno tiene 14 pines de entrada / salida digital (de los cuales 6 pueden usarse como salidas PWM), 6 pines de entrada analógica, una conexión USB, un conector de barril de alimentación, una cabecera ICSP y un botón de reinicio.

 

Cómo utilizar el tablero Arduino

Los 14 pines de entrada / salida digital se pueden usar como pines de entrada o salida usando las funciones pinMode (), digitalRead () y digitalWrite () en la programación de arduino. Cada pin funciona a 5 V y puede proporcionar o recibir una corriente máxima de 40 mA, y tiene una resistencia de arranque interna de 20 a 50 KOhms que se desconecta de forma predeterminada. De estos 14 pines, algunos pines tienen funciones específicas como se enumeran a continuación:

  • Los pines serie 0 (Rx) y 1 (Tx): los pines Rx y Tx se utilizan para recibir y transmitir datos seriales TTL. Se conectan con el chip serie ATmega328P USB a TTL correspondiente.
  • Pines de interrupción externa 2 y 3: estos pines se pueden configurar para activar una interrupción en un valor bajo, un flanco ascendente o descendente, o un cambio en el valor.
  • Pines PWM 3, 5, 6, 9 y 11: Estos pines proporcionan una salida PWM de 8 bits mediante el uso de la función analogWrite ().
  • Patillas SPI 10 (SS), 11 (MOSI), 12 (MISO) y 13 (SCK): estas patillas se utilizan para la comunicación SPI.
  • LED incorporado Pin 13: este pin está conectado con un LED incorporado, cuando el pin 13 está ALTO - el LED está encendido y cuando el pin 13 está BAJO, está apagado.

Junto con 14 pines digitales, hay 6 pines de entrada analógica, cada uno de los cuales proporciona 10 bits de resolución, es decir, 1024 valores diferentes. Miden de 0 a 5 voltios, pero este límite puede aumentarse utilizando el pin AREF con la función de referencia analógica ().  

  • El pin analógico 4 (SDA) y el pin 5 (SCA) también se utilizan para la comunicación TWI mediante la biblioteca Wire.

Arduino Uno tiene un par de otros pines como se explica a continuación:

  • AREF: se utiliza para proporcionar voltaje de referencia para entradas analógicas con función de referencia analógica ().
  • Restablecer Pin: Haciendo este pin BAJO, reinicia el microcontrolador.

 

Comunicación

Arduino se puede utilizar para comunicarse con una computadora, otra placa Arduino u otros microcontroladores. El microcontrolador ATmega328P proporciona comunicación serial UART TTL (5V) que se puede hacer usando el pin digital 0 (Rx) y el pin digital 1 (Tx). Un ATmega16U2 en la placa canaliza esta comunicación serie a través de USB y aparece como un puerto virtual para el software en la computadora. El firmware ATmega16U2 utiliza los controladores COM USB estándar, y no se necesita un controlador externo. Sin embargo, en Windows, se requiere un archivo .inf. El software Arduino incluye un monitor en serie que permite enviar datos textuales simples hacia y desde la placa Arduino. Hay dos LED RX y TX en la placa arduino que parpadean cuando los datos se transmiten a través del chip USB a serie y la conexión USB a la computadora (no para comunicación en serie en los pines 0 y 1). Una biblioteca de SoftwareSerial permite la comunicación en serie en cualquiera de los pines digitales del Uno. El ATmega328P también admite la comunicación I2C (TWI) y SPI. El software Arduino incluye una biblioteca Wire para simplificar el uso del bus I2C.

Programación Arduino

Una vez que arduino IDE esté instalado en la computadora, conecte la tarjeta con la computadora usando un cable USB. Ahora abra el IDE arduino y elija la placa correcta seleccionando Herramientas> Placas> Arduino / Genuino Uno, y elija el Puerto correcto seleccionando Herramientas> Puerto. Arduino Uno está programado usando el lenguaje de programación Arduino basado en Wiring. Para comenzar con la placa Arduino Uno y hacer parpadear el LED incorporado, cargue el código de ejemplo seleccionando Archivos> Ejemplos> Conceptos básicos> Parpadeo. Una vez que el código de ejemplo (que también se muestra a continuación) se haya cargado en su IDE, haga clic en el botón 'subir' que se encuentra en la barra superior. Una vez que finaliza la carga, debería ver parpadear el LED incorporado del Arduino. A continuación se muestra el código de ejemplo para parpadear:

// la función de configuración se ejecuta una vez al presionar reiniciar o encender la placa
configuración vacía () {
  // inicializa el pin digital LED_BUILTIN como una salida.
  pinMode (LED_BUILTIN, SALIDA);
}

// la función de bucle se ejecuta una y otra vez para siempre
bucle de vacío () {
  digitalWrite (LED_BUILTIN, ALTO); // enciende el LED (ALTO es el nivel de voltaje)
  retraso (1000); // Espera un segundo
  digitalWrite (LED_BUILTIN, BAJO); // apaga el LED haciendo que la tensión sea BAJA
  retraso (1000); // Espera un segundo
}
